# janiyah-cutchinbrown-senior-project

## OVERVIEW
This project is a web-based calorie tracking system that will be designed to monitor daily calorie intake in a simple way. It will allow users to log meals and visualize progress over time. 

## FEATURES
-User account creation 

-Daily food intake logging with calorie values

-Calculation of total daily calories 

-Goal setting

## PLANNED STACK
-FRONTEND: React.js

-BACKEND: C# (.NET Web API)

-DATABASE: Microsoft SQL Server
